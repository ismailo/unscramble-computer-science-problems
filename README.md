# Unscramble Computer Science Problems

This project analyzes real-world communication data (calls and texts) to solve practical problems using Python and core data structures.

## Problem Overview

The dataset contains records of phone calls and text messages. The goal is to extract insights such as:

- Identifying unique phone numbers
- Detecting telemarketers
- Analyzing call durations
- Understanding communication patterns

## Files

- Task0.py – Identify unique telephone numbers
- Task1.py – Find the longest call time
- Task2.py – Analyze area codes and call patterns
- Task3.py – Detect possible telemarketers
- Task4.py – Additional analysis
- Analysis.md – Time complexity explanations
- calls.csv / texts.csv – Input datasets

## Concepts Used

- Dictionaries
- Sets
- String parsing
- Sorting
- Time complexity (Big-O)

## Key Learnings

This project strengthened my ability to:
- Work with real datasets
- Optimize for time complexity
- Break problems into smaller steps
- Use Python data structures effectively
